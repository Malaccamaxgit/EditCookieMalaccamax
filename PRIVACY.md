# Privacy Policy

> **No data collection** — All data stored locally, never transmitted.

**Last Updated:** March 2026

## Data Collection

Edit Cookie Malaccamax **does not collect, transmit, or share any user data**.

## Data Storage

All extension data is stored **locally on your device** using Chrome's storage API:

| Data Type | Storage | Encryption |
|-----------|---------|------------|
| Cookie configurations | `chrome.storage.local` | None |
| User preferences | `chrome.storage.local` | None |
| Protected cookies list | `chrome.storage.local` | None |
| Filter rules | `chrome.storage.local` | None |

**Note:** This extension manages browser cookies but does not store their values persistently. Protected cookies are stored as name/domain pairs only.

## Data Transmission

This extension **does not**:

- Send data to external servers
- Use analytics or tracking services
- Contact third-party APIs
- Include advertisements
- Collect telemetry

## Permissions Used

| Permission | Purpose |
|------------|---------|
| `cookies` | Read, modify, and delete cookies (core functionality) |
| `storage` | Save preferences and protected cookies list locally |
| `contextMenus` | Add right-click menu option to edit cookies |
| `clipboardWrite` | Copy cookies to clipboard in various formats |
| `downloads` | Export cookies to file with Save As dialog |

## Third-Party Services

None. All code runs locally within the extension. No external services are contacted.
